lottie.js

A fast, dependency-free Lottie engine for every JavaScript runtime and environment. No DOM, no eval, no native code.

GitHub npm

Install

npm install lottie.js

Quick start

import { parse, CanvasSurface } from 'lottie.js';

const anim = parse(json);
const surface = new CanvasSurface(canvas.getContext('2d'));
surface.render(anim, 30);

On a server, render straight to PNG bytes without a headless browser:

import { parse, ImageSurface } from 'lottie.js';

const png = await new ImageSurface(512, 512).png(parse(json), 0);

Why lottie.js

Zero dependencies

The core has no runtime dependencies, no node: modules, and makes no assumption about its host. It runs in browsers, Node, Deno, Bun, Web Workers, and edge runtimes.

Server rendering

A pure JavaScript rasterizer renders animations to RGBA pixels and PNG on the server. No headless browser, no native bindings.

CSP-safe

No eval, no Function constructor, no inline script tricks. It works under the strictest Content Security Policy.

Three surfaces, one scene

Canvas2D for playback, SVG strings for static output, raw pixels for servers. All three draw the same renderer-agnostic scene.

Embeddable anywhere

The evaluated scene is plain data: matrices, cubic contours, paints. Feed it to Skia in React Native, WebGL, or any custom backend.

Small and disposable

A fraction of the size of lottie-web, with a real destroy() and no leaks. Playback is a thin, clockless driver you can tick by hand.

Format coverage

The full 2D vector feature set: shapes with every modifier (trim paths, repeaters, polystars, rounded corners, offset paths with real joins, zig zag, boolean merge paths), gradient fills and strokes with dashes, masks with all seven modes plus opacity and expansion, alpha and luminance track mattes, all blend modes, motion blur, precompositions with time remapping, image layers, slots, and text layers from embedded glyphs with animators and text on a path.

Expressions plug in through an optional evaluator hook, so the core stays eval-free. Layer effects are next on the roadmap.
